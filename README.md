# 학생 vs 교수 ⚔️

## 프로젝트 소개

대학교를 배경으로 한 2D 격투 게임입니다.
학생이 되어 교수와 격투를 벌이는 게임입니다.

---

## 🎮 게임 화면

| 화면 | 설명 |
|------|------|
| 입력 화면 | 플레이어 이름, 교수님 성함, 학교 이름 입력 |
| 게임 화면 | 실시간 격투 진행 |
| 클리어 화면 | 승리 시 결과 표시 |
| 게임오버 화면 | 패배 시 결과 표시 |

---

## 🕹️ 조작법

| 키 | 동작 |
|----|------|
| `← →` | 이동 |
| `↑` | 점프 |
| `↓` | 방어/회피 |
| `Space` | 기본 공격 |
| `↑ + Space` | 점프 공격 |
| `Z` | 강공격 |
| `X` | 킥 |
| `C` | 필살기 |

---

## ⚙️ 주요 기능

- **라운드 시스템** - 다중 라운드 진행 및 라운드 승패 기록
- **HP 바** - 플레이어 / 보스 체력 실시간 표시
- **필살기 게이지** - 게이지 충전 후 필살기 발동
- **콤보 시스템** - 연속 공격 콤보 카운트 표시
- **타이머** - 라운드 제한 시간
- **데미지 숫자** - 피격 시 데미지 수치 표시
- **히트 이펙트** - 공격 적중 시 이펙트
- **방어 시스템** - 방어 성공 시 GUARD 표시
- **상태 표시** - 기절(STUN), 다운(DOWN), 방어(GUARD) 뱃지
- **화면 흔들림** - 강한 공격 시 화면 쉐이크 효과
- **체력 위험 경고** - HP 낮을 때 HP바 깜빡임
- **타이머 위험 경고** - 시간 부족 시 타이머 빨간색으로 변경

## 🎨 디자인

- **배경** - 대학교 캠퍼스 이미지 (`campus.jpeg`)
- **테마 컬러** - 레드 오렌지 (`#ff4400`) 계열
- **폰트** - `Courier New` (모노스페이스)
- **타이틀 폰트** - 나무 질감 스타일 (`Georgia` serif 계열)
- **해상도** - 반응형 (기본 `900 x 600`, 전체화면 대응)
